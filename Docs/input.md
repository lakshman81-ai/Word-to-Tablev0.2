+======================================================================+===+
| -   Design limits acc. To ASME B16.5 flange rating mat. grp 1.1 and  |   |
|     SP-2094.                                                         |   |

|                                                                      |   |

|                                                                      |   |
| -   For basis of design see DEP 31.38.01.10 -- Gen                   |   |
|                                                                      |   |
| -   Use of coated bolt sets should be considered when exposed to     |   |
|     marine environments.                                             |   |
|                                                                      |   |
| -   Piping class design meets full vacuum at ambient condition.      |   |
|                                                                      |   |
| -   Due to supply chain constraints the wall thickness for pipe      |   |
|     DN300 (NPS12) SCH20 (6.35 mm) is ordered as per MESC description |   |
|     with wall thickness 7.14mm. (Applicable for 11422 series only)   |   |
|                                                                      |   |
| -   Minimum branch size is restricted to DN 50, due to limitation in |   |
|     PE/ rotoining                                                    |   |
|                                                                      |   |
| -   Threaded branch fitting is applicable only for Annulus vent      |   |
|     connection on PE lined spools.                                   |   |
|                                                                      |   |
| -   Pre -- fabricated carbon steel spool shall have flanged end and  |   |
|     shall be limited to L, T & Z shapes as given in this piping      |   |
|     class. However, exact size of the spool shall confirm to         |   |
|     vendor's recommendation for Rotolining.                          |   |
|                                                                      |   |
| -   Gaskets are not required for the PE / Roto lined flanged         |   |
|     connections. However, Retaining rings shall be provided around   |   |
|     PE / Roto lined flanged connections in order to limit the        |   |
|     compression of PE / Roto lined flanges while bolting. Width of   |   |
|     the retaining ring shall be calculated considering 20%           |   |
|     compressibility for rotolining and 15% for PE liner.             |   |
|                                                                      |   |
| -   DN 50 Monoflange slimline valves shall be selected for clean     |   |
|     services only.                                                   |   |
|                                                                      |   |
| -   In the assemblies, Monoflange valve with 10mm bore fitted with   |   |
|     3/8\" compression tube fitting is listed based on PDO Instrument |   |
|     Hook-up diagram.                                                 |   |
|                                                                      |   |
| -   The use of red. bore ball valves is considered most economical.  |   |
|     Use full bore ball valves only when necessary for                |   |
|     process/operating requirements                                   |   |
|                                                                      |   |
| -   The use of triple eccentric butterfly valves and dual plate      |   |
|     check valves is considered most economical.                      |   |
|                                                                      |   |
| -   Use gate valves and swing check valves only where fouling and    |   |
|     abrasion are negatively affecting their functionality            |   |
|                                                                      |   |
| -   HDPE / Rotolining shall be in accordance with SP-2094 and DEP    |   |
|     31.40.30.34-Gen                                                  |   |
|                                                                      |   |
| -   PE liner thickness provided in this piping class is preliminary  |   |
|     considering 3 bar collapsible pressure for Looser Liner type.    |   |
|     However, vendor shall confirm / calculate wall thickness for PE  |   |
|     Liner as per SP-2094 for the applicable liner type.              |   |
|                                                                      |   |
| -   Thickness of the stub end at flanged ends shall be twice the     |   |
|     thickness of the liner                                           |   |
